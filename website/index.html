Configuration

Intégration en 30 secondes

Ajoutez le serveur à votre configuration MCP habituelle (Claude Desktop ou Cursor).

// Dans votre fichier configuration mcp.json
{
  "mcpServers": {
    "ffbb": {
      "command": "npx",
      "args": ["-y", "@smithery/sdk-mcp-ffbb-api-client"]
    }
  }
}
